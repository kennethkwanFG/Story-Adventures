How to Add Images

📁 Character Sprites

  1. Save your character images
  2. Rename them exactly: jin.png, suga.png, jhope.png, rm.png, jimin.png, v.png, jungkook.png
  3. Copy to: assets/characters/
  4. Refresh this page to use them
✅ Supported: JPG, PNG, GIF, WebP
❌ HEIC does NOT work (convert to JPG/PNG first)

🖼️ Background Images

  1. Save your background images with any filename (e.g., kitchen.jpg, bedroom.png)
  2. Copy to: assets/backgrounds/
  3. When editing a scene, type the filename in the "Background Image" field
💡 Example: If you have living_room.jpg in assets/backgrounds/, type "living_room.jpg" in the scene editor
💡 Pro Tip: Start building your story first without images. You can add art later by just typing the filenames!
